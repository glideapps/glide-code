# Component Selectors

Common elements when working with components in Layout Editor.

## Component Picker

After clicking "+" to add component:

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Search input | Filter components | Search input at top |
| Categories | Component groups | Category headings |
| Component items | Available components | Clickable items |

### Category Names

| Category | Components |
|----------|------------|
| AI | Custom (Beta) |
| Title | Simple, Image, Profile, Cover |
| Collections | Card, List, Table, Data Grid, Checklist, Calendar, Kanban, Custom, Comments, Chat |
| Layout | Container, Separator, Tabs Container, Spacer |
| Text | Text, Notes, Rich Text, Hint, Headline |
| Content | Fields, Location, Image, Video, Big Numbers, Progress, Audio, Map, Charts |
| Action | Button Block, Link, Action Row, Rating, Button, Contact |
| Form | Contact Form, Form Container |
| Form Elements | Text Entry, Date Time, Number Entry, etc. |
| Advanced | Web Embed, Breadcrumbs, Scanner, Signature, Spinner, Tabs |

## Component Configuration (Right Panel)

### Common Elements

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| General tab | Basic settings | Tab "General" |
| Options tab | Additional settings | Tab "Options" |
| Data dropdown | Column binding | Dropdown with columns |
| Label input | Display label | Input for label/title |
| Visibility toggle | Show/hide conditions | "Visibility" section |

### Title Components (Profile, Cover, Image)

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Image source | Image column | Dropdown for image |
| Title | Main text | Input or dropdown for title |
| Subtitle | Secondary text | Input or dropdown |

### Big Numbers

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Data | Number column | Dropdown |
| Label | Description | Input field |
| Format | Number format | Format options |

### Contact Component

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Email | Email column | Dropdown |
| Phone | Phone column | Dropdown |

### Collection Components

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Style selector | Collection style | Dropdown or buttons |
| Data source | Relation or table | Dropdown |
| Title field | Main text column | Dropdown |
| Subtitle field | Secondary text | Dropdown |
| Image field | Image column | Dropdown |
| Columns (Table) | Visible columns | Column list |

### Container Component

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Columns | Number of columns | 1/2/3/4 selector |
| Add to container | Put components inside | Components list within container |

### Form Elements

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Column | Target column | Dropdown |
| Label | Field label | Input |
| Placeholder | Hint text | Input |
| Required | Make required | Toggle |

## Actions Configuration

### Actions Section

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Actions list | Current actions | List of actions |
| Add action | New action | "Add action" or "+" |
| Action item | Individual action | Item in list |
| Remove action | Delete action | X or delete button |

### Action Types (in picker)

| Action | Description |
|--------|-------------|
| Show New Screen | Navigate to screen |
| Show Form Screen | Open form |
| Go to Tab | Switch tab |
| Add Row | Create record |
| Set Values | Update data |
| Delete Row | Remove record |
| Open Link | Open URL |
| Compose Email | Start email |
| Compose SMS | Start text |
| Phone Call | Start call |
| Copy to Clipboard | Copy text |
| Show Notification | Display message |

### Action Configuration

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Target | What to act on | Dropdown |
| Column (Set Values) | Which column | Dropdown |
| Value | New value | Input or dropdown |
| URL (Open Link) | Web address | Input |
| Email fields | To/Subject/Body | Inputs |

## Visibility Conditions

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Add condition | New condition | "Add condition" |
| Column | What to check | Dropdown |
| Operator | Comparison | Dropdown |
| Value | Compare to | Input |
| And/Or | Combine conditions | Toggle/dropdown |
