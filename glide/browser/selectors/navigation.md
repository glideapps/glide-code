# Navigation Selectors

Common elements in Glide's main navigation.

## Main Navigation Tabs

These appear at the top of the builder.

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Data tab | Opens Data Editor | Look for text "Data" in main nav |
| Layout tab | Opens Layout Editor | Look for text "Layout" in main nav |
| Workflows tab | Opens Workflows | Look for text "Workflows" in main nav |
| Settings tab | Opens Settings | Look for text "Settings" in main nav |

**Note:** Don't use the "Agent" tab if present - it's error-prone.

## Dashboard Elements

On go.glideapps.com (main dashboard):

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| New app button | Creates new app | Text "New app" or "+ New app" |
| App cards | Existing apps | Cards with app names/icons |
| Team selector | Switch teams | Dropdown with team name |

## App Creation Modal

After clicking "New app":

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Blank option | Create empty app | Text "Blank" in template list |
| Import file | DO NOT USE | Text "Import a file" |
| Create button | Confirms creation | Text "Create app" or "Create" |
| Cancel button | Closes modal | Text "Cancel" or X button |

## Layout Editor - Left Sidebar

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Navigation section | Main tabs area | Text "Navigation" heading |
| Menu section | Slide-out menu items | Text "Menu" heading |
| Add tab (+) button | Adds new screen | "+" button near Navigation/Menu |
| Screen/Tab items | Individual screens | Items under Navigation/Menu |
| Components section | Screen components | Text "Components" or component list |
| Add component (+) | Opens component picker | "+" in Components area |

## Layout Editor - Screen Type Menu

After clicking add tab (+):

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Screen from data | Collection backed by table | Text "Screen from data" |
| Custom screen | Blank screen | Text "Custom screen" |
| Form screen | Data entry form | Text "Form screen" |

## Layout Editor - Preview Area

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Device switcher | Phone/tablet/desktop | Icon buttons in preview toolbar |
| Preview content | The app preview | Main preview area |
| Items in collection | Clickable to go to detail | Items in the preview |

## Layout Editor - Right Panel

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Component config | Settings for selected | Right panel content |
| General tab | Basic settings | Text "General" tab |
| Options tab | Advanced settings | Text "Options" tab |
| Actions tab | Action configuration | Text "Actions" tab |
| Data bindings | Column selectors | Dropdowns with column names |
| Save button | Saves changes | Text "Save" if present |

## Settings Page

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Name input | App name | Input in "Name & Icon" section |
| Icon button | Opens emoji picker | Icon/emoji near name input |
| Upload button | Custom icon upload | Text "Upload" |
| Color picker | Accent color | Color swatch or input |
| Layout selector | Left/Top nav | Radio/buttons for layout |
| Theme selector | Light/Deep/Accent | Radio/buttons for theme |

## Emoji Picker (Emoji Mart)

| Element | Description | Finding Strategy |
|---------|-------------|------------------|
| Search input | Filter emojis | Input at top of picker |
| Category tabs | Emoji categories | Icons/tabs in picker |
| Emoji buttons | Individual emojis | Button with emoji |
| Close/outside | Closes picker | Click outside picker |
